# ItemStack

ItemStack（物品堆栈）是一个物品数据结构，用于表示玩家或世界中的物品。每个物品都存储在一个ItemStack对象中。ItemStack包含有关物品的信息，如物品的类型、数量、附魔、耐久性等。

### 方法
|  方法名   | 类型  | 参数  |
|  ----  | ----  | ----  |
|  get_damage   | int  | void |
|  get_count   | int  | void |
|  get_display_name   | string  | void |
|  get_name   | string  | void |
|  is_empty | boolean | void |
